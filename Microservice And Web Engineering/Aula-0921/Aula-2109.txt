 Para essa aula, já criou um sistema em dotnet que é como se fosse uma cadastro de produto. No momento, essa aplicação tem só uma funcionalidade
 e ela tem um problema de segurança nessa funcionalidade.
 Vamos subir esse sistema usando Docker:
 
 1. clonei o repositorio do professor
 2. coloquei na minha pasta
 3. abri o local da pasta e no explorador dentro da pasta eu apaguei o que estava escrito e digitei cmd para abrir já dentro da pasta
 4. com o cmd aberto eu digitei "code ." e apertei tab para ele procurar pelo projeto, com o projeto selecionado eu apertei enter para ele abrir o projeto no vscode.
 5. iniciar o servico do Docker
 6. iniciar o Docker Service
 7. Abrir no vscode o terminal, entrar na pasta misc e digitar "docker compose up" para baixar a imagem
 8. apos baixar a imagem, podemos acessar localhost:8080
 9. Acessamos a pagina inserindo as informações
 10. Ao lado esquerdo entrar na parte "Command SQL"
 11. E inserir os comendos ali do banco:

 - exemplo 2: SELECT * FROM "Product" WHERE "Name" like 'mer%'--%'
